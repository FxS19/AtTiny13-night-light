# AtTiny13-night-light
Simple testcircuit bassed on a AtTiny13 with two LEDs and a 5v super cap

Features:
* Efficient Standby to charge via solar
* Bright LED 5 Minutes
* Bright LED 10 Minutes
* Bright LED 15 Minutes
* Candle Mode until mower runs out
* Cycle between modes with a debounced button press
